# :bulb: Objective
The purpose of this repository is educational. I created this material to facilitate access to knowledge. The resources provided here are designed to help learners with their first steps in the subjects.

# :books: Contents
+ **dm_betacydextrin**: This manual describes how to perform a molecular dynamics simulation of the β-cyclodextrin in an aqueous system. It was originally developed to guide an undergraduate research student in the [Computational Chemistry Laboratory (LaQC)](https://www.instagram.com/laqc_unifei/) at the Federal University of Itajubá, Brazil.
